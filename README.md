# Canon iR2425 — Бот-Оркестратор v7.0

Управление печатью с телефона через Telegram.
Отправил PDF → настроил кнопками → получаешь live-статус прямо в чате.

---

## Установка

```bash
pip install pypdf pysnmp-lextudio aiohttp aiogram
```

**Windows:** положите `SumatraPDF.exe` рядом со скриптами.

---

## Настройка (5 минут)

### 1. Создайте бота
1. Напишите [@BotFather](https://t.me/BotFather) → `/newbot`
2. Получите **TOKEN** — впишите в `config.ini`:
   ```ini
   [telegram]
   token = 123456789:ABCdefGHI...
   ```

### 2. Узнайте ваш chat_id
1. Напишите боту любое сообщение
2. Откройте в браузере:
   ```
   https://api.telegram.org/bot<TOKEN>/getUpdates
   ```
3. Найдите `"chat":{"id": XXXXXX}` — впишите в конфиг:
   ```ini
   chat_id = 987654321
   ```

### 3. Укажите IP и имя принтера
```ini
[printer]
ip   = 192.168.1.100
name = Canon iR2425
```

**Как найти IP:** На панели Canon: Меню → Сеть → TCP/IP → IP-адрес

**Как найти имя принтера:**
- Windows: Пуск → Принтеры → правая кнопка → Свойства → имя вверху
- Linux: `lpstat -p`

### 4. Запуск
```bash
python bot.py
```

## Использование

### Отправка задания
1. Откройте бота в Telegram
2. Отправьте PDF-файл (до 20 МБ через Telegram API)
3. Настройте параметры кнопками:

```
┌─────────────────────────────────────────┐
│ 📄 document.pdf                          │
│ 📊 Страниц: 240                          │
│                                          │
│ 📋 Параметры:                            │
│   • Диапазон:   1–240 (все)             │
│   • Дуплекс:    длинная ↔️              │
│   • Копий:      1                        │
│   • Пачка:      60 стр. (~30 листов)    │
│   • Охлаждение: 25 мин                   │
│   • Нагрузка:   ~30 листов/пачку        │
├─────────────────────────────────────────┤
│ [↔️ Длинная ✅] [↕️ Короткая] [📄 Нет] │
│ [📑 Страницы: все (240)] [✏️ Диапазон] │
│ [➖] [📑 Копий: 1]    [➕]      [✏️]   │
│ [➖] [📦 Пачка: 60 стр.] [➕]   [✏️]  │
│ [➖] [🌡 Пауза: 25 мин]  [➕]   [✏️]  │
│         [🚀 Запустить печать]           │
│               [❌ Отмена]               │
└─────────────────────────────────────────┘
```

4. Нажмите **🚀 Запустить**

### Live-статус во время печати

Сообщение обновляется автоматически каждые 15 секунд:

```
🖨 Пачка 2/4
[████████████░░░░░░░░]

📄 Страницы: 0061–0120
📨 Отправляю задание на принтер...
📊 Счётчик: 1847 → ожидаем +62
```

```
✅ Пачка 2/4 напечатана
[██████████░░░░░░░░░░]

📊 Напечатано: 62 стр.

🌡 Охлаждение принтера:
[████████░░░░░░░░░░░░] 18:42
⏰ Следующая пачка в 14:35
```

### Команды
| Команда | Действие |
|---|---|
| `/start` | Начало работы |
| `/status` | Статус принтера + текущее задание |
| `/stop` | Остановить печать после текущей пачки |
| `/cancel` | Отменить настройку задания |

## Настройки принтера и ноутбука

Для стабильной работы длинных сессий (7–10 часов):

**Canon iR2425:**
- Увеличьте время ухода в спящий режим до **240 мин** или выключите
- Путь: Меню → Параметры → Таймер/Энергосбережение

**Windows (ноутбук/ПК с ботом):**
- Выключите режим сна: Пуск → Электропитание → «Никогда» при питании от сети
- Иначе Python-процесс остановится вместе со сном

## Ограничения Telegram API

- Для файлов > 20 МБ: положите PDF на ПК с ботом и используйте CLI-режим (`python main.py file.pdf`)
